# Zod-AR

**Arabic** translations of **Zod** validations message

## Quick Start
```cli
yarn add zod @mhmdaljefri/zod-ar
```

Inside the application

```jsx
import { arErrorMap } from "@mhmdaljefri/zod-ar";
import * as z from "zod";

// rest of code

z.setErrorMap(arErrorMap);

// rest of you code
```
